# Supply Chain & Inventory Optimization Analysis 📦

## 🎯 Business Objective
The goal of this project is to optimize logistics operations by analyzing freight costs, warehouse capacities, and order distribution. By connecting multiple data sources, I aim to identify cost-saving opportunities and improve delivery efficiency.

## 📊 Data Architecture
This project utilizes a Relational Database structure with 7 core tables:
* OrderList: Transactional data for customer orders.
* FreightRates: Cost analysis per carrier and shipping mode.
* WhCapacities & WhCosts: Warehouse operational metrics and rental costs.
* ProductsPerPlant: Manufacturing distribution data.
* VmiCustomers & PlantPorts: Logistics and priority customer mapping.

## 🛠️ Tech Stack
* Excel: Initial data cleaning and format conversion (XLSX to CSV).
* MySQL: Relational schema design and advanced SQL Joins.
* Power BI: Interactive dashboarding for stakeholder insights.
